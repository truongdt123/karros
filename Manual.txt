This project is using maven for building and running with Java 8.
1. Build and run Unit Test  
  1.1 Go to project folder :
  1.2 Execute : mvn compile test
  - The report of coverage unit test will be place at target\site\jacoco
 -  Open index.html or jacoco.cvs
 2.  Run project  
  - Go to project folder
  - Execute: mvn spring-boot:run
  Go to the link as below for Swagger API:
  http://localhost:8080/swagger-ui.html
